Research about custome properties

  • To define a custom Property's name, we have to use two hyphens(--) place before the name. We also can use hyphen(-) or underscore(_) characters.
  • We can use all characters except Unicode(U+0040) in custom Property name.
  • The possible value that we can assign to a new Custome Property is a value that using a var() function.
  • It is useful to creat Custom Properties using the :root or html selector when we want to select html with a higher specificity. However,
  • After we've defined a new custome Property, to recall that Custome Property, we use the the calc() function.
  • The ways to override a Custom Property are:
    • text color
    • Header
    • Button
  • Custom Properties are very useful to avoid repeatation in CSS and also it can help us to simplify styling various part(theming) in our css.
  • The only aspect (or part) of css syntax that we can use Custom Properties is inside of declaration block.
