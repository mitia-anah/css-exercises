  • We can use all characters except a periode or fullstop in custom Property name.
  • The possible value that we can assign to a new Custome Property is a number, normal text.

  • Custom Properties are very useful to avoid repeatation in CSS and also it can help us to simplify styling various part(theming) in our css. It is a one point of control, which means making changes in one place.
