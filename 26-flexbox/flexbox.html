Flexbox

  1. The order of property values in the shorthand flex property is flex_flow.

  2. In flex property
    : 2, 1 and auto represent:
    • 2 :variation.
    • 1 A number specifying how much the item will grow relative to the rest of the flexible items.
    • auto the size of the items based on its width and height properties. or its content if nnot set.

  3. Before we can use align_content, we set the values of flex_wrap or flex_flow to be inline-flex.

  4. The difference between align_item and justify_content is that:
    The align-items property defines the default behavior for how items are laid out along the cross axis while
    justify-content aligns individual items within the main-axis.

  5. Align_self is a th property which overrides the other container.

  6. The other commonly-used box model that similarly to flex-basis is box_sizing

  7. There are two ways to change the ordering of flex items within a flex container.
    One is done on each flex item by using the `order` property. The other can change the whole order of
    the flex items at the container level. The property that we need is flex dirrection.

  8. If we want some spaces between each of our flex items,wich is in a flex column,
    we will use align-content and space-around to achieve that.
