Overflow and Clipping

1 st part

Overflow css property allows us to control the contents to be fit in its block formatting content. See those properties and their default values bellow:

  • Overflow: visible(default), hidden, scroll, auto
  • Clip: auto
  • Clip-path: none
  • Text-overflow: ellipsis

It is very important to use the overflow property to control the content when it is too big to fit the box and we also can use clip property when we want to define what portion of an element is visible. Then we use the text-overflow to know how overflowed the content is. In addition, we also use the clip-path to specify a part of an element which we want to show.

Another properties which we need to be set before these properties are the overflow-y and overflow-x properties.

2 nd part

The browser's default way to handle contents that are larger than its container is `450pixel`

An ellipsis is a key word value to represent clipped text but the ellipse decrease the text displayed.

Text overflow can only work on block or inline-block level elements, because the element needs to have a width in order to be overflow-ed.
