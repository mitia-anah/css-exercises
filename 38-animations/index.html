Keyframe animations

  1. Keyframe Animation setup needs some unique code, which always starts with @Keyframes
  2. What makes the keyframes code different to the rest of CSS is that it has curly bracket.
  3. To use a keyframe animation we set up the name of the animation as a property in a ruleset for an element.
  4. Infinite, alternate, alternate-reverse, animation-iteration, play-state and posed-state are the keywords that aren't available in transition.
  5. We can animate many properties
  6. The anitable properties are:
    1. Background
    2. Borders
    3. bottom
    4. Clip
    5. colors
    6. columns/rows
    7. Flex
    8. Fonts
    9. Grids
    10. Margins
    11. Paddings
    12. Transform
    13. Visibility
    14. Width/height
    15. Z-index
