floats and block formatting contexts

Float and clear properties:

  1. The default values for float and clear are none.
  2. These are the possible values for clear:
    • right
    • left
    • none
    • both
  3. The rule associated with floated, in terms of source order is that:
    • the element floated must come before an other element
    • to make text or an element to wrap arround them
  4. Can we float other elements besides image? Yes we can!
  5. There is a way to create a two-column layout with float,
    which is separating the contents into two part by using a div for example.

BFC (Block formatting contexts)

  1. As my understand, BFC is all about transforming the layout of block boxes and sets of rules for how box behaves in differently containers.

  2. We can 'clear' a float by creating a block formatting context on the parent,
    this is achieved by putting a certain set of properties on the container. The properties are:

    • overflow: hidden, auto, etc
    • display: inline-block
