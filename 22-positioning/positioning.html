Positioning

Position property

Positioning is a layout-breaking form of layout. i.e. it pulls elements out of the normal flow, and one has to use known dimensions to account for these positioned elements.

the default value for Position is STATIC.

There are some other properties that we need to supply when using an non-default Position value, which are the :

  • top
  • Bottom
  • left
  • right

When a value of an element has been positioned, the inferred display value is absolute.

The one of the non-default values for positioning ,which still maintains a box in the flow of the document is the relative value

The special about fixed and sticky is that:

  • sticky position can keep the content's normal flow size.
  • sticky position establishe a new containing block for absolutely positioned descendants
  • fixed positioned boxes do not move when the document is scrolled.
  • in fixed position the containing block is established by the viewport.

What makes absolute positioned useful are:

  • to specify position
  • to specify dimension

The CSS's stacking contex is the Z-attribute, which allows the user to render the order of layering of boxes.
