Table styling

  • Table-layout

    • The table-layout has only two defaults values, which are `Fixed` and `Auto`.

  • Border-spacing

    • The default value that we can get from border-spacing is that the `length value`, which also specifies size of a vertical or horizontal spacing between cells.

  • Border-collapse

    • In the Border-collapse, we have those two different values, which are `separate value` and `collapse value`. We use the ``separate value` to make a space between two border which is inside of the table we currently work on. In that case, each cells might have their own border. But for the `collapse value`, we have different value. The table is just normal. In othe word, there is no separation of any border anymore seen inside of the table.

  • Caption-side

    • Caption-side has varities default values that set a position in our table. They are top, bottom, block-start, block-end, inline-start and inline-end.

  • Empty-cell

    • Empty-cell is quite interesting because we can make our value to be appear or hidden by using these two possible values, which called `Show` and `Hide`
