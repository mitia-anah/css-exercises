Research about text-shadow and box-shadow

  1. box-shadow and text-shadow have very similar value syntax but the only different is we cannot apply thickness value on text-shadow but only with the box-shadow.
  2. You should use the combination of x, y, thickness, colour if you want to creat hard-edge shadow.
  3. It is possible to create shadow on a circular element, yes we can use with poligone as well.
  4. Yes , it is possible to creat multiple shadows on the same element.
