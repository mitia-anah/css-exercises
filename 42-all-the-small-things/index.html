All the small things

Important but small properties

Cursor

  1. Keyword value/fallback( pointer,auto). Value(URL)/ it uses the structure of url(...),url(...)with comma separated list.
  2. There are category of cursors:
    • General: default,-none,-auto
    • Link and status contex-menu,help,pointer,progress,wait.
    • Selection: cell,crosshair,text,vertical-text.
    • Drag and drop: alias,copy,move,no-drop,not-allowed,grab,grabbing,
    • Resizing and scrolling: all-scroll, col-resize,row-resize,etc
    • Zooming: zoom-in, zoom-out
  3. The cursor property is specified as zero or more url values, separated by commas, followed by a single mandatory keyword value.

Pointer-event

  1. Property and value: pointer-events:
    • None
    • auto
    • visiblePainted
    • visibleFill
    • visibleStroke
    • painted
    • fill
    • stroke
    • all
    • visible
  2. Pointer-event can support with every browser.
  3. The pointer-events property defines whether or not an element reacts to pointer events.
  4. If we want to have the element respond to pointer events,we set the value to (auto), or set it to (none) if we want to prevent it from doing so.

Outline

  1. The Outline property is a shorthand to set various outline properties. These properties draw a line arround the outside of the element:

    • Outline-style: sets the style of the outline.
    • Outline-color: the color of the outline.
    • Outline-offset:
    • Outline-width: the thickness of the outline.

Filter

  1. Applies graphical effect to an element. It is also used to arrange something like images or background or bordersomitted sub-values.
  2. There are several filter-function values. For instance, blur(),brigthness(),contrast,drop-shadow,sepia,saturate,etc.
  3. We can use multiple filter at one time by using comma to separate the value.
